Couchback Demo

To what goes on here, open up your browser console. Feel free to explore the functionality by inspecting the source.

To see sync happen, sign up and then sign in.

Sign up

Signing up creates a user in remote PouchDB

Couchback.signUp(username, password, callback)

Sign in

Signing in authenticates the user and returns a url to the remote pouchdb

Couchback.signIn(username, password, callback)

We have now set up the PouchDB sync.

So, we'll try inserting random objects to test the sync

myPouchDB.put(randomObject, callback)
